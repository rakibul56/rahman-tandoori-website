## Setup

1. Alle Dateien in einen Webserver-Ordner kopieren
2. **Logo-Dateien hinzufügen**:
   - `logo.png` - Logo für Navigation, Hero-Bereich und Footer
   - `tranparent_logo.png` - Transparentes Logo für Hintergrund-Effekte
3. `index.html` in einem Browser öffnen
4. Für lokale Entwicklung kann ein einfacher HTTP-Server verwendet werden:

```bash
# Python 3
python -m http.server 8000

# Node.js (mit http-server)
npx http-server

# VS Code Live Server Extension
```

## Logo-Implementierung

Die Website verwendet ein cleveres Logo-System für optimale Darstellung:

- **logo.png**: Verwendet als sichtbares Logo in Navigation und Footer
- **tranparent_logo.png**: Verwendet als subtiler Hintergrund-Effekt (Wasserzeichen-Stil) im Hero-Bereich

### Logo-Spezifikationen

- **Format**: PNG mit transparentem Hintergrund empfohlen
- **Größe**: Mindestens 200x200 Pixel für scharfe Darstellung
- **Sichtbares Logo**:
  - Navigation: 50x50px (42px auf Tablet, 35px auf Mobile)
  - Footer: 45x45px
- **Hintergrund-Logo**: 800x800px (subtiles Wasserzeichen im Hero und Services-Bereich)
